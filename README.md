## Faça buscas mais eficientes, utilizando arvores binárias em PHP

Com o BSTPhp você pode realizar buscas mais eficientes em sua base dados, onde estará utilizando todos os benefícios de uma arvore binária de buscas, extraida do material de uma biblioteca em java, chamada algs4.
